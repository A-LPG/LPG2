TypeScript GLR forest (demo)

    

      In-browser parse with the TypeScript GLRParser (GSS/SPPF) over the ambiguous
      E ::= E + E | NUMBER grammar. Root nextAst alts and SPPF symbol-node
      counts show packing / sharing (n=8 trees Catalan 429, SPPF nodes ≪ 429).
      Java can be generated via WASM with GLR checked, but cannot drive parse in the browser.
