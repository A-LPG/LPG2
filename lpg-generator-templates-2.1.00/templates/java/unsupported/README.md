# Unsupported Java templates

Legacy A–E (and early F) `.g` templates kept for historical reference only.

**Do not use** these in new grammars. Active templates are the `*F.gi` files in
the parent directory (`dtParserTemplateF.gi`, `btParserTemplateF.gi`,
`glrParserTemplateF.gi`, `LexerTemplateF.gi`, …).
